# C++ Practice Library

A collection of small C++ programs for practicing fundamentals, file handling, classes, and debugging.

## Topics

- Variables and input/output
- Conditionals and loops
- Functions
- Arrays and vectors
- File input/output
- Classes and objects
- Debugging practice

## Build

Use a C++17 compiler:

```bash
g++ -std=c++17 src/main.cpp -o practice
./practice
```

Or use the included Makefile:

```bash
make
./build/practice
```

## Project Goals

This repo shows consistent practice, readable code, and improvement over time.
